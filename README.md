Актуальные рабочие MTProto proxy для Telegram.  
На странице собраны бесплатные прокси-серверы для быстрого подключения к Telegram в один клик.

Сайт со списком proxy:

https://fauster740.github.io/TelegramProxyRU/

## Бесплатные proxy для Telegram

Если Telegram плохо открывается, долго подключается или нестабильно работает через мобильный интернет, можно попробовать бесплатные MTProto proxy.

На сайте собраны прокси для Telegram по странам:

- Финляндия;
- Швеция;
- Германия.

## Что есть на сайте

- бесплатные proxy для Telegram;
- рабочие MTProto-серверы;
- быстрые ссылки для подключения;
- инструкция по настройке proxy в Telegram;
- подключение прокси в один клик.

## Как подключить proxy в Telegram

1. Откройте сайт со списком proxy.
2. Выберите подходящий сервер.
3. Нажмите кнопку подключения.
4. Подтвердите подключение proxy в Telegram.

Список прокси может обновляться. Если один сервер не работает, попробуйте другой.
